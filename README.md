PROGRAMOVÁNÍ

Filtrování: Program projde celý seznam a vybere jen vyhovující řádky. Zahodí tyto objednávky:  nejsou zaplacené (FALSE), mají prázdný název nebo mají místo ceny nesmyslný text (třeba "abc"). 

Seskupování (sčítání): Protože jeden zákazník může mít v jedné objednávce více věcí (např. myš a klávesnici), program tyto položky najde podle čísla objednávky a sečte jejich ceny dohromady. 

Řazení a výpis: Nakonec všechny hotové objednávky seřadí od té nejdražší po nejlevnější a p vypíše je pod sebe do terminálu.
